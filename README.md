# Atris Plugins

Plugin marketplace for [Claude Code](https://claude.com/claude-code) and Cowork.

## Install

In Claude Code or Cowork:

1. Open **Plugins** > **Discover** > **Add Marketplace**
2. Enter: `atrislabs/atris-plugins`
3. Browse and install plugins

## Available Plugins

### atris-workspace

AI workspace operating system with 15 skills: Gmail, Google Calendar, Google Drive, Slack, Notion, Google Slides, plus writing, design, backend, and project management tools.
